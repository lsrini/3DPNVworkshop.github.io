# 3DPNVworkshop.github.io

#Editing the content

Everything is hand-editable in index.html:

Date / venue — in the hero section (.hero-meta) and the Location section.
Schedule — each talk is a .t-item block under #morning / #afternoon.
Status pills use the classes tag-ok (Confirmed), tag-tbc (To be confirmed), tag-tba (TBA).
Organizers — .org-card blocks; avatars are the speaker's initials.
Partners — .partner chips in the footer.
Map — the <iframe> in the Location section uses a Google Maps embed pointed at ENSTA Paris.
